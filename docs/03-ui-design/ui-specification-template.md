# UI仕様書 - {画面名}

> **使用方法**: このテンプレートをコピーして、各画面のUI仕様書を作成してください。
> {中括弧}内の項目は実際の内容に置き換えてください。
> **セクションの統一**: すべての画面でセクション構成を統一するため、該当しないセクションも削除せず「該当なし」と記載してください。

---

**重要:** このUI仕様書を作成する前に、必ず [UI共通仕様書](./ui-common-specification.md) を参照してください。UI共通仕様書には、すべての画面で共通するUI仕様（ボタン、フォーム要素、テーブル、モーダル、バリデーション、エラーメッセージ、アクセシビリティ等）が定義されています。このテンプレートには、**画面固有の仕様のみ**を記載してください。

## 要素概要

**注:** レイアウト構成で定義した各要素について、汎用的な項目を一覧で記載します。詳細は「要素詳細」セクションを参照してください。

| No | 項目名 | オブジェクト名 | 入力タイプ | I/O | 必須 | 最大桁 | 初期値 | 表示内容 | 備考 |
|------|------|------|------|------|------|------|------|------|------|
| (1) | ヘッダー | header | - | O | - | - | - | 固定値: 画面タイトル | 固定表示 |
| (2.1) | キーワード | keyword | Text | I/O | - | 255 | - | - | 部分一致検索 |
| (2.2) | 検索ボタン | search_button | Button | I | - | - | - | 固定値: 「検索」 | プライマリボタン |
| (3.1) | 新規作成ボタン | create_button | Button | I | - | - | - | 固定値: 「+ 新規作成」 | 権限制御あり |
| (4.1) | 名前 | name | Link | O | - | - | - | DB: `Entity.name` | ソート可 |
| (4.2) | アクションボタン | action_buttons | Button | I | - | - | - | 固定値: 「詳細」「編集」 | テキストリンク |

**列の説明:**
- **No**: 要素番号（詳細セクションの番号と対応）
- **項目名**: 画面上の表示名または要素の役割
- **オブジェクト名**: プログラム内で使用する識別子（変数名、フィールド名等）
- **入力タイプ**: Text, Button, Label, Link, Select, MultiSelect, Radio, Checkbox, Badge, Alert, Pagination等
- **I/O**: I（入力）、O（出力）、I/O（入出力）
- **必須**: ○（必須）、-（任意）
- **最大桁**: 最大文字数・桁数（HTML maxlength属性による物理的な入力制限）
- **初期値**: デフォルト値
- **表示内容**: 表示するデータやラベルのソース（`固定値:` / `DB:` / `前画面:` / `設定ファイル:` の接頭辞で記載）
- **備考**: その他の重要な情報

**注:** 最大桁はHTML maxlength属性による物理的な入力制限を示します。バリデーションルールの詳細は「要素詳細」セクションに記載してください。
