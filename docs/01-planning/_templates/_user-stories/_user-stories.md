# ユーザーストーリー一覧 テンプレート

> **使い方**:このテンプレートは `user-stories/user-stories.md` の記述ルールを定義します。
> - 各セクションの「記述ルール」と「記述例」を参考にして内容を記述してください。
> - **出力形式**: 実際に作成するファイルには「記述ルール」と「記述例」は含めず、内容のみを記述してください。
> - **セクションの統一**: 該当しないセクションも削除せず「該当なし」と記載してください。

## ユーザーストーリー一覧

### 記述ルール

- 全ユーザーストーリーを一覧表形式で記載する
- 機能カテゴリでグループ化する
- ファイル名は `US{機能ID下3桁}-{機能名(英語)をkebab-case}.md` 形式とする
- 機能IDと機能名(英語)は [functional-requirements.md](../functional-requirements.md) を参照
- 個別ユーザーストーリーの記述は [_user-story.md](./_user-story.md) を参照

### ディレクトリ構造

#### 記述例

```
user-stories/
├── user-stories.md
├── US001-login.md
├── US002-logout.md
├── US003-password-reset.md
├── US010-order-create.md
├── US011-order-list.md
...
```
