## このディレクトリについて

このディレクトリには、UIとAPIの相互作用（インタラクション）に関する詳細設計ドキュメントが格納されます。

**格納する内容:**
- UI×API紐づけ（マッピング）
- 処理フロー・シーケンス図
- バリデーション実行タイミング
- エラーハンドリングフロー
- 状態管理
- パフォーマンス最適化

**注:**
- UI設計（画面レイアウト、バリデーションルール定義）は `docs/03-ui-design` ディレクトリで管理されています。
- API仕様（エンドポイント、リクエスト/レスポンス）は `docs/04-api` ディレクトリで管理されています。
- このディレクトリは、UI設計とAPI設計が概ね固まった後に作成する**詳細設計**に該当します。

## ディレクトリ構成

```
05-interactions/
├── README.md                              # このファイル
├── interaction-specification-template.md  # インタラクション仕様書テンプレート
│
├── asset-management/                      # 資産管理機能
│   ├── asset-search/
│   │   └── interaction-specification.md   # 資産検索画面のインタラクション仕様
│   ├── asset-detail/
│   │   └── interaction-specification.md   # 資産詳細画面のインタラクション仕様
│   └── ...
│
├── lending-management/                    # 貸出管理機能
│   ├── lending-application/
│   │   └── interaction-specification.md   # 貸出申請画面のインタラクション仕様
│   └── ...
│
└── ...                                    # その他の機能
```

**注:** ディレクトリ構成は `docs/03-ui-design` と同じ構造（機能カテゴリ/画面名）で管理します。
