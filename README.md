# Claude Code Spec Development Template

このレポジトリは、Claude Codeを使用した仕様駆動開発を行う際のテンプレート資料を格納したものです。

## 目次

- [目的](#目的)
- [使用方法](#使用方法)
- [ドキュメント管理について](#ドキュメント管理について)
- [ライセンス](#ライセンス)

## 目的

新規プロジェクトを開始する際に、このレポジトリをコピーして使用することで、効率的にAIによるコーディングを進めることができます。各種ドキュメントを参照させながら、一貫性のある開発を実現します。

## 使用方法

1. このレポジトリを新規プロジェクトのディレクトリにコピー
2. プロジェクトの要件に合わせて各ドキュメントを編集
3. Claude Codeにドキュメントを参照させながら開発を進行
4. 必要に応じてドキュメントを更新・追加

ドキュメントの詳細な構成と作成ガイドラインは [docs/README.md](docs/README.md) を参照してください。

## ドキュメント管理について

このプロジェクトは **Gitによるバージョン管理** を前提としています。

- ドキュメント内に改訂履歴や変更履歴を記載する必要はありません
- すべての変更履歴は **Gitのコミット履歴** で管理されます
- ドキュメントは常に最新の状態を保ち、過去のバージョン情報はGitで参照してください

## ライセンス

プロジェクトに応じて適切なライセンスを設定してください。
